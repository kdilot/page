## 패키지 설치

> yarn add -D @svgr/webpack

## 코드설정

### next.config.js

```ts
webpack: (config) => {
  config.module.rules.push({
    test: /\.svg$/,
    use: ['@svgr/webpack'],
  });
  return config;
},
```

### example

```ts
import SampleSvg from './images/github.svg'

return <SampleSvg />
```
