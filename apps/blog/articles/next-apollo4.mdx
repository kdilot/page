## 패키지 설치

> yarn add -D @graphql-codegen/cli @graphql-codegen/typescript

## 설정 추가

### package.json

```ts
...
  "scripts": {
    ...
    "generate-gql-types": "graphql-codegen"
  },
...
```

### codegen.yml

```yml
schema: http://localhost:3200/api/graphql
generates:
  types.ts:
    plugins:
      - typescript
```

### run

```ts
yarn dev
yarn generate-gql-types
```
