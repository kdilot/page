## 패키지 설치

depenencies

> yarn add @emotion/react @emotion/styled

devDependencies

> yarn add --dev @emotion/babel-plugin

## 설정 추가

### .babelrc

```ts
{
  "presets": [
    [
      "next/babel",
      {
        "preset-react": {
          "runtime": "automatic",
          "importSource": "@emotion/react"
        }
      }
    ]
  ],
  "plugins": ["@emotion/babel-plugin"]
}
```
